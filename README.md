# Online Shopping System Database (SQL Project)

This project is a relational database designed for an Online Shopping System. 
It manages data for Customers, Products, Orders, Payments, and Order Details.

## Features
- Database & Table Creation
- Primary & Foreign Keys
- Insert, Update, Delete
- Aggregate Queries
- Joins
- Subqueries
- Case Statements
- Data Validation

## Technology
- MySQL

## Purpose
This project demonstrates SQL concepts for database design and management.
